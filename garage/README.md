# The Blueprint Shelf

This is where your project ideas live before they become real tools.

Each blueprint is a ready-to-use prompt. You hand it to Jarvis
(your AI orchestrator) and it walks you through building it step by step.
You don't need to know how to code. You just need to know what problem you want to solve.

## How to Use a Blueprint

1. Open the blueprint file
2. Copy the prompt under "Hand This to Jarvis"
3. Paste it into Claude Code
4. Answer Jarvis's questions — it will tell you what it's going to build before it writes anything

That's it. Your first build usually takes 1–2 hours. After that you'll have
something you can use tomorrow morning.

---

## Building Something Not on This List?

Tell Jarvis what problem you're trying to solve:

> "I have a problem I want to build a tool for. Here's what's happening:
> [describe the problem in plain English]. Walk me through building
> something to fix it — start with a plan before any code."

Jarvis will write the plan. You approve it. Then the build starts.
