# My Projects — Wolf Pack Builder

This folder is where you build things.

Each project gets its own subfolder. Start simple — one problem at a time.

---

## Ideas to Get You Started

When you're ready to build, ask Claude to help you create a new project folder.
Here are some good first projects for property managers:

### Quick Wins (1–2 hours each)
- **lease-tracker** — A list of all your leases sorted by expiration date
- **late-rent-log** — Track who owes what and when they paid
- **maintenance-log** — Log requests and mark them resolved
- **vendor-contacts** — Directory of your plumbers, electricians, etc.

### Medium Projects (a few sessions)
- **renewal-reminder** — Auto-email tenants 60 and 30 days before lease ends
- **owner-report-generator** — Fill in numbers, get a formatted monthly report
- **move-in-checklist** — Digital checklist with photo notes per unit
- **vacancy-tracker** — Track vacant units, days on market, showings

### Bigger Builds (a few weeks)
- **tenant-portal** — A simple web page tenants can use to submit requests
- **owner-dashboard** — A private page each owner can log into to see their properties
- **automated-notices** — Send late rent warnings automatically

## How to Start a New Project

Tell Claude:

> "I want to start a new project. Create a folder called [project-name] in
> the projects directory. Then ask me what I want it to do before writing
> any code."

Claude will set up the folder and work with you from there.
